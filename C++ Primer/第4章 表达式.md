# 第4章 表达式
## 算数运算符
```
+ -                // 一元正负号 
* / % + -
```
## 逻辑和关系运算符
```
! < <= > >= == != && ||
```
## 赋值运算符
```
=
```
## 递增和递减运算符
```
++ --
```
## 成员访问运算符
```
ptr->mem
(*ptr).mem
```
## 条件运算符
```
? :
```
## 位运算符
```
~ << >> & ^ |
```
## sizeof运算符
```
sizeof(type)       // 返回类型所占字节数
sizeof expr        // 返回表达式结果类型的大小
// 返回size_t类型
```
## 逗号运算符
```
先执行左侧表达式，再执行右侧表达式，并返回右侧表达式结果
```
## 类型转换
隐式类型转换
* 比int小的整型提升为int（整型提升）
* 在条件中，非布尔值转换为布尔值
* 初始化中，初始值转换为变量的类型
* 算数运算或关系运算中，转换为同一类型
* 函数调用时，发生类型转换（见第6章）
## 运算符优先级
```
算术 > 位 > 身份 >成员 > 比较 > 逻辑 > 赋值
```
